title: nexus

## Installation

```sh
npx @uplift-ltd/nexus

yarn global add @uplift-ltd/nexus
```

## API

### build-library

Builds `cjs` and `esm` folders using typescript.

```
nexus build-library
```

### clean-library

Removes the `cjs` and `esm` folders (if they are not checked into git).

```
nexus clean-library
```
